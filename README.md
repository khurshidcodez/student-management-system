# Student Management System

A console based Student Management System built using C.

## Features

1. Add student records.
2. Display all students.
3. Search student by roll number.
4. Update student details.
5. Delete student records.
6. File storage using text files.

## Technologies and Concept Used

1. C Programming.
2. File Handling.
3. Structures.
4. Arrays
5. Git/GitHub.

## How to Run

1. Clone repository
2. Compile:
   gcc student.c -o student
3. Run:
   ./student
